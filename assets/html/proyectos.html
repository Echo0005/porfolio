App InterLinked

Trabajo grupal con estudiantes autodidactas.

La App consiste en una plataforma pensada para facilitar la union entre Desarrolladores, Emprendedores e Inversores, para llevar adelante proyectos.

Usamos Angular para el Front-End y Spring Boot junto a MySQL para el Back-End. Dado nuestros tiempos entre otros trabajos, estudios y prioridades, el proyecto ha tenido sus pausas y relentizaciones.
